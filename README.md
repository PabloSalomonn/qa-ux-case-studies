📱 QA UX Case Studies – Mobile Testing Portfolio
📌 Descripción

Este repositorio contiene casos reales de testing manual enfocados en experiencia de usuario (UX) en aplicaciones móviles.

Los casos fueron desarrollados a partir de testing exploratorio, identificación de problemas de interacción y documentación de bugs con enfoque en impacto de negocio.

🎯 Objetivo

Demostrar habilidades en:

Testing manual
UX testing
Análisis de interacción en mobile
Documentación profesional de bugs
Pensamiento orientado a producto

🧩 Enfoque de testing
Testing exploratorio

💡 Insight clave

Los problemas detectados no son solo bugs funcionales, sino oportunidades de mejora en la experiencia de usuario con impacto directo en conversión y retención.
