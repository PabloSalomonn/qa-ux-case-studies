 🐞 Bugs documentados

1. Activación accidental "Agregar al carrito"

- 📱 Dispositivos:
  - iPhone 13
  - Samsung Galaxy A17 5G

- 🔍 Tipo:
  - UX Issue / Functional Bug

- ⚠️ Problema:
  El botón "Agregar al carrito" se activa accidentalmente durante el scroll en listados de productos.

- 💥 Impacto:
  - Fricción en la experiencia
  - Acciones no intencionales
  - Interrupción del flujo de compra


🧪 Test Cases

Se incluyen casos de prueba diseñados para validar:
- Interacciones táctiles
- Comportamiento durante scroll
- Prevención de errores de usuario


 📊 Enfoque de testing

Este proyecto aplica:
- Exploratory Testing

- Cross-device testing


📎 Evidencia
