# Test Cases - Add to Cart UX

## TC-01: Tap accidental durante scroll

**Objetivo:**
Validar que no se agreguen productos sin intención

**Pasos:**
1. Abrir listado
2. Hacer scroll continuo

**Resultado esperado:**
No se activan botones

---

## TC-02: Validación de zona táctil

**Objetivo:**
Validar tamaño y ubicación del botón

**Resultado esperado:**
Botón no interfiere con scroll

---

## TC-03: Recuperación de error

**Objetivo:**
Validar opción de deshacer

**Resultado esperado:**
Usuario puede revertir acción fácilmente
